# User Story: 10 - Join WhatsApp Group

**As a** confirmed passenger,
**I want** to join the trip's WhatsApp group,
**so that** I can communicate with the driver and other passengers.

## Acceptance Criteria

* "Join on WhatsApp" button visible after booking
* Each trip has unique WhatsApp group link
* Button opens WhatsApp with group join link
* Button disabled when trip is full
* Full trips show "Waitlist" status
* Clear messaging about group purpose

## Notes

* WhatsApp is primary communication channel
* Consider waitlist notification system
