# User Story: 2 - View Trip Itinerary

**As a** potential passenger,
**I want** to view a detailed itinerary for each trip,
**so that** I can understand the trip plan and activities before joining.

## Acceptance Criteria

* "View Itinerary" button appears on each trip card
* Button opens a modal or page showing the full itinerary
* Itinerary displays daily activities in chronological order
* Each activity shows time, location, and description
* Itinerary is rendered from stored JSON data
* Modal/page is responsive and readable on all devices

## Notes

* Itinerary data comes from the trip creation flow
* Button appears alongside "Join on WhatsApp" with proper mobile stacking
