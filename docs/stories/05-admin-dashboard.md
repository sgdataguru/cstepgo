# User Story: 5 - Admin Dashboard

**As an** administrator,
**I want** to access a dashboard with user activity and system metrics,
**so that** I can monitor platform usage and ensure system health.

## Acceptance Criteria
* Admin can view aggregated user activity data
* System health metrics are displayed
* Admin can filter data by date and user segment

## Notes
- Access control required for admin features
- Data privacy and security must be maintained
