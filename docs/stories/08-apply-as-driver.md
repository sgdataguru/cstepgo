# User Story: 8 - Apply as Driver

**As a** potential driver,
**I want** to apply through a structured onboarding process,
**so that** I can become verified and start offering trips on the platform.

## Acceptance Criteria

* Multi-step application flow:
  1. Initial application form
  2. Document upload (license, insurance, etc.)
  3. Verification pending status
  4. Onboarding materials/training
  5. Go live confirmation
* Progress indicator shows current step
* Can save and resume application
* Clear requirements for each document
* Notification when verification complete

## Notes

* Thorough verification needed for safety
* Consider background check integration
