# User Story: 23 - Driver Dashboard Interface

**As a** driver,
**I want** a dedicated dashboard showing my earnings, trip statistics, and current status,
**so that** I can track my performance and manage my driving activity effectively.

## Acceptance Criteria

* Dashboard displays daily, weekly, and monthly earnings
* Shows completed trips count and average rating
* Current online/offline status toggle
* Today's trip statistics (trips completed, hours online, earnings per hour)
* Trip history with customer ratings and feedback
* Earnings breakdown showing platform commission and net earnings

## Notes

* Serves as the main landing page for drivers after login
* Should provide clear overview of driver performance metrics
* Earnings display must account for platform commission (15-25%)
* Need to include tax reporting features for driver compliance
