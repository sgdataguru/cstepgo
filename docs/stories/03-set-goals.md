# User Story: 3 - Set Step Goals

**As a** user,
**I want** to set daily and weekly step goals,
**so that** I can challenge myself and track my progress towards healthier habits.

## Acceptance Criteria
* User can set custom step goals
* Progress towards goals is displayed visually
* Notifications are sent when goals are achieved or missed

## Notes
- Goal data should be persistent across sessions
- Option to adjust goals as needed
