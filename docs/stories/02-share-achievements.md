# User Story: 2 - Share Achievements

**As a** user,
**I want** to share my step achievements with friends or on social media,
**so that** I can celebrate my progress and encourage others to join me in staying active.

## Acceptance Criteria
* User can share step milestones via supported social platforms
* Shared content includes step count and achievement badge
* User can select which achievements to share

## Notes
- Social media integration required
- User consent for sharing data must be obtained
