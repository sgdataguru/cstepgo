# User Story: 6 - View Driver Profile

**As a** potential passenger,
**I want** to view detailed driver profiles,
**so that** I can build trust and confidence before booking a trip.

## Acceptance Criteria

* Driver name and photo displayed on trip cards
* Clicking driver info opens full profile page
* Profile includes:
  - Years of driving experience
  - Vehicle information
  - Ratings and reviews
  - Languages spoken
  - Verification badges
  - List of upcoming trips
* Profile page is publicly accessible
* Responsive design for all devices

## Notes

* Driver profiles are critical for building trust
* Verification badges need clear meaning
